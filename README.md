# Personal Financial Analyst

Self-hosted personal finance stack: ingest statements into PostgreSQL, drive budgeting and analytics, and chat with a backend-run agent—see the **[product requirements document](docs/PRD.md)** for full scope.

**Living PRD (discussion):** [Issue #1 — PRD: Personal Finance MCP server + agent-driven UI](https://github.com/chandra-jagarlamudi/PersonalFinancialAnalyst/issues/1)

**Roadmap:** [GitHub Issues](https://github.com/chandra-jagarlamudi/PersonalFinancialAnalyst/issues) (vertical slices).

---

Slice **0** (Postgres via Docker Compose) is tracked in **[Issue #2](https://github.com/chandra-jagarlamudi/PersonalFinancialAnalyst/issues/2)**. Implementations and PRs are linked from that issue; use **`main`** (default branch) as the source of truth for **`compose.yaml`**, **`.env.example`**, and **`docker compose up`** once those files exist in your checkout.
